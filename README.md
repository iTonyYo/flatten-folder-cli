# $ flatten-folder

递归扁平化指定文件夹内所有文件。

## 安装

```shell
# 使用 NPM
$ npm i -g flatten-folder-cli

# 使用 Yarn
$ yarn global add flatten-folder-cli
```

## 使用

```
使用方式
  $ flatten-folder 选项 [...]

选项
  --twd, -d, 目标目录，默认：'process.cwd()'
  --version, -V, 查看版本号
  --help, -h, 查看帮助

示例
  $ flatten-folder
```
